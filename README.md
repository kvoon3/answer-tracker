# Answer Tracker

A dead simple, keyboard-friendly answer tracking application

## Features

- **Multiple Answer Modes**: Track answers, input correct answers, and compare results
- **Keyboard Navigation**: Vim-style shortcuts for efficient navigation
- **Auto-scrolling**: Automatically scrolls to keep focused questions visible
- **Progress Tracking**: Real-time statistics on completion and accuracy
- **Responsive Design**: Works on both desktop and mobile devices
- **Modal System**: Help and settings accessible via modals
- **Importing**: Import user answers and standard answers
